## Setup
download or clone the repo
--run
```bash
$ npm install
```
-import database
```bash
 import the db file from the database folder
```
-setup env
```bash
    rename .env.example to .env
    Replace your DB and Port details
    Save your file
```
-start
```bash
 $   npm start
```
